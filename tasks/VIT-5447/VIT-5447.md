Название кода при клонировании шаблона

## Описание проблемы

При клонировании шаблона заявлении копия создается с кодом XXX_COPY, однако в коде поддерживаются только alphanumeric. Из-за этого нет возможности сохранить дупликат без изменения кода и удаления запрещенных символов.



## Описание задачи

При создании дупликата код нового заявления генерируется без "_" — XXXCOPY.



## Acceptance Criteria

* [ ] После клонирования шаблона заявлений код копии не содержит в себе запрещенных символов, только цифры и буквы
* [ ] Сохранение сразу после клонирования происходит успешно
